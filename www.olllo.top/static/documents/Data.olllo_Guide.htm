Data.olllo Super Data Processing Function

I have to say the super function, including super filter, super extraction, super replacement. The so-called super, is super.

Support ordinary strings, but also support the rule description method (regular expressions). For regular expressions, it provides a function to draw expressions, so that people who do not want to learn regular expressions can use them.

With this super function, what the find, filter, select, replace, match, etc, etc, etc can be done.

1. Super Filter: You can filter the selected column, the results of the filter exists in a new field of this data table.

For example, you want to see if there is a cell phone number in a category, then select the regular expression for cell phone number,

and then click OK, the results will be saved, the results of True, it means that there is a cell phone number, False, it means no.

If you want to extract the whole dataframe, then use the "filtering records" function to determine whether it is equal to True or equal to False, it can be.

2. Super Extraction: You can extract the content of the selected column, the results exist in a new field of this data table. For example, to extract the cell phone number of a field column, then select the regular expression for the cell phone number, and then OK, the result The result will be extracted into a new column (you can name it).

3. Super extraction, you can also use the content as a split keyword. For example, if I want to split a field by number (the numbers are not the same), then select the content of the split in Super Extraction, and then select the number regular expression, After confirming, you can get a new column with the segmented content.

4. Super Replacement: You can replace the matching content with what you want. For example, you want to replace the cell phone number with 139xxxxxx, then select the column, then select the regular expression for cell phone, fill in the replacement content as "139xxxxxx", and then OK, the replacement result will exist in a new column, look for it.

5. Super Function is also integrated with the graphical generation of regular expressions, there are examples, see and try and understand.
